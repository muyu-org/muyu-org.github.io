VSCode

全称Microsoft Visual Studio Code，懂了？写 MarkDown 文档只是牛刀小试，还可以用它写 Latex、Python、C/C++等等，几乎所有的语言。你只需要下载它，首先安装上简体中文包，并切换为简体中文，再安上 Markdown Preview Enhanced 插件就可以了。开源的！开源的！！开源的！！！

Atom

全世界最大的开源平台GitHub出品的编辑器 Atom。Atom 和 VSCode 一样，并不只是用来写 MarkDown，但它依然是一款非常出色的 MarkDown 编辑器。
优点
是 GitHub 官方出品。大厂出品，品质和服务都是有保障的。
支持实时预览，这一点非常重要了。
可以安装多种插件，极大的扩展了软件的功能。各种快捷方式，神奇操作还不信手拈来。
跨平台，这点的重要性不用我多说了吧。
缺点
有点耗资源，这从启动速度就可以看得出来，不过只是启动慢点而已，只要电脑不是太慢，码字还是非常棒的！
跟 VSCode 一样，对新手不太友好。但只要肯钻研，都是有回报的。
点击官网链接了解更多。

Vim

终于写到 Vim了，我前面也说过，自从用了 Vim 后，其他编辑器我再也没正眼瞧过一眼，除了 Vim 化的 VSCode。

移动端
现代人在手机上花的时间越来越多了，所以手机排版的重要性就越来越突出了，虽然手机更加适合用来阅读，但有时候我们身边没有电脑时，想临时记一下灵感，这还是可以的。下面对我用过的一些软件进行说明，排名不分先后。

MarkdownX
MarkdownX 是一款 Android 下的 Markdown 编辑器，可以通过菜单栏按钮快速输入格式，左滑实时预览，支持插入图片，自动保存。MarkdownX 默认并不显示广告，你有需求可以开启广告，非常良心，这是一款免费应用。
