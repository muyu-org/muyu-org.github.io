Obsidian之静态站点生成

作者：木鱼精灵
时间：2023年10月13日 （星期五） 01:40:10

Obsidian插件

  1. 安装Webpage HTML Export，Obsidian具备静态化能力。
  2. 新建专门的文件夹，用于存放自己对外发布的Blog，建议通过模板功能制作Blog的模板，该文件夹的md文档使用。
  3. 根据需要输出静态站点，同步到服务器。

插件名称：Webpage HTML Export
插件作者：Nathan George
插件说明：将文档、文件夹或库导出成 HTML 文件，有黑白主题以及大纲目录。
项目地址：https://github.com/KosmosisDire/obsidian-webpage-export
————————————————
直接来源地址： PKMer

阿里云OSS

  1. 在阿里云购买对象存储OSS，新建Bucket，设置静态页面托管、CNAME绑定。
  2. 考虑安装阿里云自己的终端软件OSS-Browser。
  3. 配置RAM 访问控制，获取终端登录信息。
  4. 同步Obsidian的静态文件夹到服务器。

网上说，阿里云的OSS香港节点有5G免费存储和免费流量，正好有阿里云账号，开通一个试试。
帮助手册：https://help.aliyun.com/zh/oss/

详解之一：https://forum-zh.obsidian.md/t/topic/10274/1

仅标准存储（本地冗余）容量支持使用5 GB/月的免费额度（即每月标准存储（本地冗余）容量≤5 GB时，不收取标准存储（本地冗余）容量费用）。支持使用免费额度的地域为：中国香港等。

仅外网流出流量（NetworkOut）支持使用5 GB/月的免费额度（即每月外网流出流量≤5 GB时，不收取外网流出流量费用），支持使用免费额度的地域为：中国香港等。

评论区有人推荐：国内就用OSS是最优选，如果使用国外的服务，研究AWS不如研究 Cloudflare R2 54。存储空间免费10G，下载流量全免费，就是网络有时候也不太行。
《十个国内优秀对象云存储服务使用对比》

阿里云的香港节点，每月有免费的5GB存储空间和5GB下载流量。
也就是说，如果你的Obsidian库不超过5G，同时你每个月的同步下载流量也不超过5G，那么使用香港节点的阿里云是完全免费的。
如果你的库大小超过了5G，那么可以购买一个香港的存储空间资源包，也是9元/40G/年。这样的话，每年只需9元，你就得到了一个存储空间40G，下载流量60G的阿里云OSS服务，这对一般的笔记来说已经是绰绰有余了。

①中国内地各地域绑定的域名必须在工信部备案，其他地域的域名绑定不需要在工信部备案。
②一个域名只能绑定在一个存储空间上，一个存储空间最多可以绑定100个域名。
③一个账号可绑定的域名个数无限制。

阿里云OSS还是很香的，我的付费虚拟主机对绑定域名的数量都有苛刻的限制，而OSS可以绑定的域名几乎算是不限制。所以，如果将@的A记录或者CNAME记录解析到阿里云，其实就可以通过文件夹形式实现多主题的静态页面。

问题及解决：
导出高级选项：
Include file tree
adds an interactive file tree to the sidebar.
可以自定义是否在左边栏显示目录树。
这个目录树如果是单独的，更新最新文档时同步刷新目录树，那么可以极大的减少文档生成数量。
但目录树如果是直接生成在文档页，那么旧文档也的目录树是没有之后文档的链接的，价值减少很多。如果要体现左边栏价值，那么Webpage HTML Export输出需要全目录输出。
我的做法是在该目录下生成一个index文档，并且用dataview动态列出本目录的目录树，输出静态页时，就是本目录的索引页。更新时，就可以只更新index和当前文档。
当然，文档模板必须有一个链接是本目录的索引页。
