阿里云OSS
...

  1. 在阿里云购买对象存储OSS，新建Bucket，设置静态页面托管、CNAME绑定。
  2. 考虑安装阿里云自己的终端软件OSS-Browser。

详解之一：https://forum-zh.obsidian.md/t/topic/10274/1
阿里云的香港节点，每月有免费的5GB存储空间 50和下载流量 33。
也就是说，如果你的obsidian库不超过5G，同时你每个月的同步下载流量也不超过5G，那么使用香港节点的阿里云是完全免费的。
同时，如果你的库大小超过了5G，那么可以购买一个香港的存储空间资源包，也是9元/40G/年。这样的话，每年只需9元，你就得到了一个存储空间40G，下载流量60G的阿里云OSS服务，这对一般的笔记来说已经是绰绰有余了。

问题及解决：
导出高级选项：
Include file tree
adds an interactive file tree to the sidebar.
可以自定义是否在左边栏显示目录树。
这个目录树如果是单独的，更新最新文档时同步刷新目录树，那么可以极大的减少文档生成数量。
但目录树如果是直接生成在文档页，那么旧文档也的目录树是没有之后文档的链接的，价值减少很多。如果要体现左边栏价值，那么Webpage HTML Export输出需要全目录输出。
我的做法是在该目录下生成一个index文档，并且用dataview动态列出本目录的目录树，输出静态页时，就是本目录的索引页。更新时，就可以只更新index和当前文档。
当然，文档模板必须有一个链接是范围本目录的索引页。
